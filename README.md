## Getting Started

First, install the required dependencies:

```bash
npm i
```

Then, start the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


## Dependencies

React Query
Moment
Zod
Tailwind
Axios

## Dates

Must be parsed with the moment library
